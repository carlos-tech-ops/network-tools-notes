Cheat sheet and notes while studying for Network+ certification.

| Port | Protocol |
|------|----------|

| 25   | SMTP     |
| 53   | DNS      |
| 80   | HTTP     |
| 110  | POP3     |
| 443  | HTTPS    |
| 3389 | RDP      |
